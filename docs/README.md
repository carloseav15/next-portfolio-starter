# Design Documentation

This directory contains internal design documents that document intent, strategy, and audit findings during development. They may differ from the live site.

- `content-model.md` — Data contracts and source-of-truth layout
- `copy-deck.md` — Page-level copy strategy and positioning
- `ux-2026-spec.md` — UX principles, route intent, and accessibility targets
- `qa-checklist.md` — Release gate checklist
- `audit/` — UX audit findings, scorecard, and remediation plan
- `ai-workflow.md` — AI collaboration and accountability model
